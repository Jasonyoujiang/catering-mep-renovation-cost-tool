# 商铺机电改造方案与成本测算

面向商业项目商铺改造的本地测算工具，用于快速生成机电改造配置建议，并查询部分机电改造子项的参考造价。

## 功能概览

- 改造方案生成：根据商铺面积、业态类型或指定用电量，生成估算用电负荷、配套电缆规格等参数；餐饮业态同步输出供水、排水、排油烟和隔油池条件。
- 成本查询：通过“造价子项 + 规格型号”两级菜单查询电缆、不锈钢风管、排油烟风机、油烟净化器等参考造价。
- 批量生成：上传商铺 Excel，按现有测算逻辑批量生成每个商铺的机电条件，并下载结果表。
- 数据可扩展：电缆规格、测算规则和造价清单均独立放在 `data/` 目录，便于后续逐项完善。

## 使用方式

直接打开 `index.html` 即可使用。当前版本已把 Excel 处理组件放在 `vendor/` 目录，日常测算、模板下载和批量结果生成不依赖外网。

也可以双击桌面上的 `启动机电成本查询.command`，程序会自动在浏览器中打开。

## 桌面安装包

本项目已增加 Electron 桌面壳，可分别生成 MacBook 和 Windows 安装包。

- [下载最新版 Mac 和 Windows 安装包](https://github.com/Jasonyoujiang/catering-mep-renovation-cost-tool/releases/latest)
- [在线使用](https://jasonyoujiang.github.io/catering-mep-renovation-cost-tool/)

首次构建前安装依赖：

```bash
npm install
```

本地预览桌面程序：

```bash
npm start
```

生成 MacBook 安装包：

```bash
npm run pack:mac
```

生成 Windows 安装包：

```bash
npm run pack:win
```

一次性生成两个平台安装包：

```bash
npm run pack:all
```

安装包输出到 `dist/` 目录。MacBook 产物为 `.dmg`，Windows 产物为 `.exe` 安装程序。

当前安装包按内部自用口径构建，未接入 Apple Developer 公证和 Windows 商业代码签名证书。首次安装或打开时，系统可能出现安全提示；确认来源为本仓库构建产物后再允许打开。

如需运行测试：

```bash
npm test
```

## 批量 Excel 模板

第一版批量模块要求 Excel 表头包含：

- 商铺编号
- 楼层
- 业态类型
- 面积
- 是否启用指定用电量
- 指定用电量

程序会生成：

- 估算用电负荷
- 配套电缆规格
- 供水管径
- 排水管径
- 排油烟风量
- 占用隔油池容积
- 测算依据
- 处理状态
- 错误说明

结果表不输出风险提示，便于作为批量铺位机电条件表继续流转。

## 当前测算口径

- 用电负荷：
  - 轻餐：250 W/m2
  - 普通餐饮：400 W/m2
  - 重油烟餐饮：500 W/m2
  - 零售：120 W/m2
  - 生活服务：120 W/m2
  - 超市：150 W/m2
  - 支持勾选“指定用电量”后按输入值输出。
- 零售、生活服务和超市：第一版仅测算用电负荷和配套电缆规格。
- 排油烟风量：
  - 轻餐：25 m3/h/m2
  - 普通餐饮：32 m3/h/m2
  - 重油烟餐饮：37 m3/h/m2
- 占用隔油池容积：
  - 轻餐：0.5 m3/100m2
  - 普通餐饮：1.0 m3/100m2
  - 重油烟餐饮：1.5 m3/100m2

## 说明

本工具为商业项目机电管理的第一版测算框架，输出结果用于前期管理测算和方案复核提示。实际项目仍需结合设计图纸、设备清单、上级电源容量、现场管线路由、施工界面和当地规范要求进行复核。
