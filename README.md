# 餐饮商铺机电改造方案与成本测算

面向商业项目餐饮商铺改造的本地测算工具，用于快速生成机电改造配置建议，并查询部分机电改造子项的参考造价。

## 功能概览

- 改造方案生成：根据商铺面积、餐饮类型或指定用电量，生成估算用电负荷、配套电缆规格、供水管径、排水管径、排油烟风量、占用隔油池容积等参数。
- 成本查询：通过“造价子项 + 规格型号”两级菜单查询电缆、不锈钢风管、排油烟风机、油烟净化器等参考造价。
- 批量生成：上传商铺 Excel，按现有测算逻辑批量生成每个商铺的机电条件，并下载结果表。
- 数据可扩展：电缆规格、测算规则和造价清单均独立放在 `data/` 目录，便于后续逐项完善。

## 使用方式

直接打开 `index.html` 即可使用。

如需运行测试：

```bash
npm test
```

## 批量 Excel 模板

第一版批量模块要求 Excel 表头包含：

- 商铺编号
- 楼层
- 业态类型
- 面积
- 是否启用指定用电量
- 指定用电量

程序会生成：

- 估算用电负荷
- 配套电缆规格
- 供水管径
- 排水管径
- 排油烟风量
- 占用隔油池容积
- 测算依据
- 处理状态
- 错误说明

结果表不输出风险提示，便于作为批量铺位机电条件表继续流转。

## 当前测算口径

- 用电负荷：
  - 轻餐：250 W/m2
  - 普通餐饮：400 W/m2
  - 重油烟餐饮：500 W/m2
  - 支持勾选“指定用电量”后按输入值输出。
- 排油烟风量：
  - 轻餐：25 m3/h/m2
  - 普通餐饮：32 m3/h/m2
  - 重油烟餐饮：37 m3/h/m2
- 占用隔油池容积：
  - 轻餐：0.5 m3/100m2
  - 普通餐饮：1.0 m3/100m2
  - 重油烟餐饮：1.5 m3/100m2

## 说明

本工具为商业项目机电管理的第一版测算框架，输出结果用于前期管理测算和方案复核提示。实际项目仍需结合设计图纸、设备清单、上级电源容量、现场管线路由、施工界面和当地规范要求进行复核。
